First-time setup
1Go to supabase.com → New project → name it "MyOrgTracker"
2SQL Editor → run this schema
create table org_volume ( id bigserial primary key, entry_date date not null, user_slug text not null, month_num int, year_num int, day_num int, org_volume numeric, org_increase numeric, dv_goal numeric default 113636, dv_remaining numeric, org_daily_avg numeric, est_org_dv numeric, uplift_per_day numeric, pin_goal text, unique(entry_date, user_slug) );
3Settings → API → copy URL & anon key → paste above
